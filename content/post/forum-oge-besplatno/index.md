## Как это работает? Проще, чем кажется

Всё строится на GitHub Pages. Вам не нужно быть айтишником. Вот пошаговая инструкция:

### 5 шагов до своего форума

1. **Зарегистрируйтесь на GitHub**  
   Перейдите на [github.com](https://github.com) — это бесплатно, можно даже через школьную почту.

2. **Откройте шаблон форума**  
   Перейдите по ссылке: [https://feekool.github.io/promo-oge-3/](https://feekool.github.io/promo-oge-3/)

3. **Создайте копию**  
   Нажмите «Use this template» → «Create a new repository». Назовите его, например, `oge-math-2026` или `oge-russian-tutor`.

4. **Включите GitHub Pages**  
   Зайдите в Settings → Pages → выберите ветку `main` → Save.

5. **Готово! 🎉**  
   Подождите около 60 секунд — ваш форум будет доступен по ссылке вида `https://ваше-имя.github.io/oge-math-2026`.

## Частые вопросы

**Нужно ли знать программирование?**
Нет. Базовая установка — это 5 кликов в интерфейсе GitHub. Если захотите кастомизировать дизайн — пригодятся основы HTML/CSS, но это опционально.

**Можно ли использовать для платных курсов?**
Да. Feekool — это инструмент, а не сервис. Вы сами решаете, как монетизировать контент. Некоторые репетиторы делают публичное демо + приватный раздел для оплативших.

**Что если я хочу сменить предмет?**
Создайте новый репозиторий из того же шаблона — это займёт 2 минуты.

**А если я не хочу использовать GitHub?**
Шаблон — это просто статический HTML/CSS/JS. Его можно загрузить на любой хостинг: Netlify, Vercel или свой сервер. GitHub Pages — просто самый простой вариант.
