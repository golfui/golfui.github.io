## Как это работает? 5 простых шагов

Вам не нужно быть программистом. Всё строится на GitHub Pages — это бесплатно, надёжно и просто.

### Пошаговая инструкция

1. **Зарегистрируйтесь на GitHub**  
   Перейдите на [github.com](https://github.com) — это бесплатно.

2. **Откройте шаблон форума**  
   Перейдите по ссылке: [https://feekool.github.io/promo-ege-2/](https://feekool.github.io/promo-ege-2/)

3. **Создайте копию**  
   Нажмите «Use this template» → «Create a new repository». Назовите его, например, `ege-tutor` или `ege-math-2026`.

4. **Включите GitHub Pages**  
   Зайдите в Settings → Pages → выберите ветку `main` → Save.

5. **Готово! 🎉**  
   Подождите около 60 секунд — ваш форум будет онлайн по ссылке `https://ваше-имя.github.io/ege-tutor`.

**Можно ли использовать для платных курсов?**
Да. Вы сами решаете, как монетизировать контент. Некоторые репетиторы делают публичное демо + приватный раздел для оплативших.

**Что если я хочу сменить предмет?**
